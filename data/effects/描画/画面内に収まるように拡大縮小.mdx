# 画面内に収まるように拡大縮小

画面内に収まるように拡大縮小
|項目|形式||||
| - | - | - | - | - |
|モード|プルダウン|拡大縮小|拡大のみ|縮小のみ|
|対象|プルダウン|縦横両方向|横方向のみ|縦方向のみ|  

|項目|形式|初期値|最大値|最小値|単位|
| - | - | - | - | - | - |
|余白上下|スライダー|0.0|99999.0|-99999.0|px|
|余白左右|スライダー|0.0|99999.0|-99999.0|px|
|回転を無視|`ボタン`|無効||||

画像アイテムやテキストアイテムに使用すれば、いちいち拡大率を手動で調整する必要がなくなります。
