# メッシュ変形

メッシュ変形
|項目|形式|初期値|最大値|最小値|単位|
| - | - | - | - | - | - |
|横制御点数|スライダー|2|32|2|個|
|縦制御点数|スライダー|2|32|2|個|
|変形|特殊コントロール|(別途記載)||||
|選択中の制御点をまとめて移動|プルダウン|||||
|X|スライダー|0.0|1000.0|-1000.0|px|
|Y|スライダー|0.0|1000.0|-1000.0|px|


## メッシュ変形特有のコントロールについて
各点のずれを調整できます。
Ctrlキーを押しながら選択することでまとめて変更することも可能です。
またプレビュー画面からの変形も可能です。
