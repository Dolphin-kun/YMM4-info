# 拡大縮小

拡大縮小
|項目|形式|初期値|最大値|最小値|単位|
| - | - | - | - | - | - |
|全体|スライダー|100.0|5000.0|0.0|%|
|横方向|スライダー|100.0|5000.0|0.0|%|
|縦方向|スライダー|100.0|5000.0|0.0|%|
|ドット絵|`ボタン`|無効||||

![初期値](/assets/effects/描画/拡大縮小/初期値.png)

縦横比を変更したい場合、横方向または縦方向のスライダーを調整することで可能です。

ドット絵を有効にすると、画像などを拡大した際にぼかしがかからず、1px1pxが明確に表示されるようになります。

![ドット絵](/assets/effects/描画/拡大縮小/ドット絵.png)
