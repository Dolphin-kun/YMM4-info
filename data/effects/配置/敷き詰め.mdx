# 敷き詰め

## エフェクト
敷き詰め
|項目|初期値|最大値|最小値|単位|
| - | - | - | - | - |
|横|3|99999|1||
|縦|3|99999|1||

![初期値](/assets/effects/配置/敷き詰め/初期値.png)

## 効果
アイテムを縦と横方向に隙間なく敷き詰めます。

## 関連エフェクト
- [余白追加](/effects/描画/余白追加/)を使用すると敷き詰めたアイテムの間に余白を追加することができます。
![余白追加-敷き詰め](/assets/effects/配置/敷き詰め/余白追加-敷き詰め.png)
