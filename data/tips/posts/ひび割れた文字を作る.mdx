## やり方
映像エフェクト`砕け散る`を使用します。

|項目|砕け散る|
| - | - |
| 開始時間 | -1.0秒 |
| 再生速度 | 0% |
| 破片の大きさ | 自由 |

|項目|砕け始める位置|
| - | - |
| X,Y,Z | 自由 |

|項目|アニメーション詳細|
| - | - |
| 飛散速度 | 5% |
| 落下速度 | 0% |
| 時間差  | 100% |
| 距離影響 | 20% |
| ランダム回転 | 0% |
| ランダム方向 | 0% |

文字ごとに分割をONにすると、文字ごとに同じ形でひび割れします。
