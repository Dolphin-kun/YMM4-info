## 使用するエフェクト
[画面の複製を使う場合](#画面の複製を使う場合)
- 波紋
- 画面の複製

[画面の複製を使わない場合](#画面の複製を使わない場合)
- 波紋
- 余白追加
- (グループ制御アイテム)


## やり方
### 画面の複製を使う場合
画面の複製アイテムにエフェクトを追加します。

|画面の複製|描画|
| - | - |
| 複製元を削除 | ON |

|映像エフェクト|波紋|
| - | - |
| X,Y | 0px |
| 波長 | 1600pxから調整 |
| 振幅 | マイナスで凸, プラスで凹 |
| 周期 | 1000秒(Max) |

### 画面の複製を使わない場合
グループ制御アイテムや、エフェクトをかけたいアイテム本体に追加して下さい。  


[画面の複製を使う場合](#画面の複製を使う場合)と同じエフェクトを追加します。
|映像エフェクト|波紋|
| - | - |
| X,Y | 0px |
| 波長 | 1600pxから調整 |
| 振幅 | マイナスで凸, プラスで凹 |
| 周期 | 1000秒(Max) |

波紋の前(上)に余白追加エフェクトを追加します。
|映像エフェクト|余白追加|
| - | - |
| 上,下,左,右 | アイテムが全て表示されるように調整 |

![順番](/assets/tips/posts/魚眼レンズ風エフェクト/順番)


## 注意点
- 周期は1000秒になっていて完全に止まっている訳ではないので、どうしてもゆっくり変化していまいます。
- これは魚眼レンズ風エフェクトであり、魚眼レンズエフェクトではありません。アイテムを完全に円形にすることは、できないと思います。
