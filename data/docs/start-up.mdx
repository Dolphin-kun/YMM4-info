# YMM4を導入する
![サムネイル画像](https://ymm4-info.vercel.app/assets/tips/docs/start-up/cover.png)

ゆっくりMovieMakerとは、饅頭遣いさんが開発している動画編集ソフトです。
このページでは、YMM4のダウンロードやその際の注意点について書かれています。

## ダウンロード
YMM4の配布されているサイトである[饅頭遣いのおもちゃ箱](https://manjubox.net/ymm4/)からYMMをダウンロードします。

![YMM4ダウンロードボタン](https://ymm4-info.vercel.app/assets/tips/docs/start-up/1.png)

サイト内からこのようなダウンロードボタンを探し、YMM4もしくはYMM4Liteをダウンロードします。
YMM4とYMM4Liteの違いは[こちら](#YMM4とYMMLiteの違い)

### YMM4とYMM4Liteの違い



## インストール
ダウンロードしたファイルを展開します。
```md:展開したフォルダの中
···
│
├ ○○.md
├ ○○.md
├ ○○.md
├ ゆっくりMovieMaker4.exe
│
···
```

インストールが終わったらフォルダ内を確認して、ゆっくりMovieMaker4.exe(もしくはゆっくりMovieMaker4Lite.exe)があることを確認してください。
![YMM4ダウンロードボタン](https://ymm4-info.vercel.app/assets/tips/docs/start-up/2.png)

> フォルダ内にずっと置いておくのは非効率的なのでデスクトップにショートカットを作成してしまうのをお勧めします。
