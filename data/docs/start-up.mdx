# YMM4を導入する
![サムネイル画像](https://github.com/Dolphin-kun/YMM4-info/blob/3bb521d7a77d3dd08af52e59c36281a6dfa27112/public/assets/tips/posts/start-up/cover.png)

ゆっくりMovieMakerとは、饅頭遣いさんが開発している動画編集ソフトです。

以下ゆっくりMovieMakerをYMMと表します。

ーーーーなにか書くーーーー

## ダウンロード
YMM4の配布されているサイトである[饅頭遣いのおもちゃ箱](https://manjubox.net/ymm4/)からYMMをダウンロードします。

YMM4とYMM4Liteの違いは[こちら](#)

## インストール
ダウンロードしたファイルを展開します。
```.md 展開したフォルダの中
···
│
├ ○○.md
├ ○○.md
├ ○○.md
├ ゆっくりMovieMaker4.exe
│
···
```

インストールが終わったらフォルダ内からゆっくりMovieMaker4.exe(もしくはゆっくりMovieMaker4Lite.exe)から起動することが出来ます。
