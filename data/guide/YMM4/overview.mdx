# YMM4とは？
ゆっくりMovieMaker4という饅頭遣いさんが開発している、ゆっくり実況プレイ動画用の動画編集ソフトで、  
YMM4と略して使われることが多いです。(このサイトでは、ゆっくりMovieMaker4をYMM4と書いています。)

## ソフトの読み方
YMM4は

- 「ゆっくりムービーメーカー・フォー」
- 「ゆっくりむーびーめいかー・よん」

のどちらでも読むことが出来ますが、開発者の饅頭遣いさんは[こちらのポスト](https://x.com/manju_summoner/status/1718234862019330222)で「よん」と読む派であるとポストしていました。

```tweet

```

## 特徴
### 無料
個人的な利用であれば、無料で使用することができます。
ただし、商標利用やそれを考えている場合であれば、ライセンスを購入する必要があります。
AquesTalk(主にゆっくりボイスと言われているもの)を必要としない場合は、YMM4Liteを使用することでライセンスを購入する必要がなくなります。
> 参考 : [Ymm4とYMM4Liteの違いは？(饅頭遣いのおもちゃ箱)](https://manjubox.net/ymm4/faq/etc/YMM4Lite/)

## 更新頻度が高い
毎月1日には機能追加などのアップデートが公開され、バグや不具合が見つかるとすぐに修正アップデートが入ります。
そのため、「YMM4を開くと必ずアップデートがある」というYMM4ユーザーの投稿を見かけることがあります。

### 操作性がよい
編集初心者にもわかりやすいUI(User Interface（ユーザーインターフェイス）)で、直感的に操作することができます。

さらに、ショートカットキーや台本機能を使いこなせるようになると、編集スピードが格段に早くなります。
> 参考 : [メイン画面](https://ymm4-info.net/guide/%E5%9F%BA%E6%9C%AC%E6%A9%9F%E8%83%BD/%E3%83%A1%E3%82%A4%E3%83%B3%E7%94%BB%E9%9D%A2/)
> 参考 : [台本ファイルをもとにボイスアイテムを追加する(饅頭遣いのおもちゃ箱)](https://manjubox.net/ymm4/faq/editing/%E5%8F%B0%E6%9C%AC%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E3%82%92%E3%82%82%E3%81%A8%E3%81%AB%E3%83%9C%E3%82%A4%E3%82%B9%E3%82%A2%E3%82%A4%E3%83%86%E3%83%A0%E3%82%92%E8%BF%BD%E5%8A%A0%E3%81%99%E3%82%8B/)

### ゆっくり実況・合成音声実況が作りやすい
ゆっくり実況などの合成音声実況が、他のソフトより簡単に作成することができます。
YMM4は、対応している音声合成ソフト・サービスが20種類以上あり、さまざまな音声を使用することができます。
> 参考 : [ゆっくりボイス(饅頭遣いのおもちゃ箱)](https://manjubox.net/ymm4/faq/%E3%82%86%E3%81%A3%E3%81%8F%E3%82%8A%E3%83%9C%E3%82%A4%E3%82%B9/)

### YMM4で作成したプロジェクトファイルをAviUtlに持ち込める
YMM4で音声部分を作成して、AviUtlで細かい編集をすることもできます。
