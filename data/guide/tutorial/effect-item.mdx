# エフェクトアイテムの使い方

![ エフェクトアイテム ](/assets/tips/posts/画面の外側だけ/エフェクトアイテム.png)

画面のうちの一部分だけエフェクトを適用させることが出来るオブジェクトです。  
従来の画面やアイテムにかけるエフェクトでは、  
画面全体またはアイテムにしかエフェクトを適用することが出来ませんが、  
このエフェクトにより、 **「画面の中のここだけ色調変えたいな～」** という願望がかなえられるようになります。  
[このような](https://ymm4-info.vercel.app/tips/posts/Syuhenboke/)表現も可能となります。(サイト内別記事へ飛びます。)

# エフェクトアイテムの設定項目
![エフェクトアイテム特有の設定項目](/assets/tutorial/effectitemstettings.png)

**ここではエフェクトアイテム特有の設定項目のみに触れています。下記以外のものが分からない場合は[「図形アイテムの使い方」](https://ymm4-info.vercel.app/guide/tutorial/shape-item/)へ**

### 1.「種類」
![エフェクトアイテム-種類](/assets/tutorial/effectitem-kinds.png)  
上の写真のように、**背景,円,三角形,五角形,六角形,星,扇形,くさび形,矢印,スーパー多角形,集中線,タイマー,波形,svgファイル,PSDファイル**
が読み込めるようになっています。それぞれの項目の使い方は[「図形アイテムの使い方」](https://ymm4-info.vercel.app/guide/tutorial/shape-item/)へ

### 2.「ぼかし」
エフェクトアイテムの効果が及ぶ範囲をどの程度ぼかすか、という項目です。

・ぼかし値0の時
![ぼかし0](/assets/tutorial/bokashi0.png)
・ぼかし値50の時
![ぼかし50](/assets/tutorial/bokashi50.png)

また、図形アイテムとぼかしエフェクトで代用して効果範囲を可視化すると、  
・ぼかし値0の時
![図形ぼかし0](/assets/tutorial/zukeibokashi.png)
・ぼかし値50の時
![図形ぼかし0](/assets/tutorial/zukeibokashi50.png)
の黒色(#000000)でない部分全てが効果範囲となります。

**ただし、エフェクトアイテムに、エフェクト「ぼかし」をかけても上のようにはなりません。効果範囲内のものがぼかされるだけです。**
